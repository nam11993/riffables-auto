## Cách dùng link này

Link hiện tại là trang danh sách issue, chưa phải một PRD cụ thể.

Bước tiếp theo:

1. Mở link GitHub issues index.
2. Chọn issue tương ứng với feature cần test.
3. Copy link issue cụ thể, ví dụ:

```text
https://github.com/speedrun-labs/riffables-prd/issues/1
```

4. Tạo file PRD source riêng cho feature đó trong thư mục này.
5. Dùng prompt `../prompts/01-requirement-analyst.md` để tạo requirement.

## Gợi ý đặt tên file sau khi chọn issue

```text
<feature-name>.prd.link.md
```

Ví dụ:

```text
login.prd.link.md
song-upload.prd.link.md
playlist.prd.link.md
```
