# PRD Change Review Reports

Thư mục này lưu report do AI tạo sau khi phân tích PRD issue changes và update tài liệu.

Luồng đúng:

```text
sync-prd-issues.ps1
-> impact report
-> AI Impact Update Assistant
-> update requirements/test plan/test cases
-> PRD change QA review report
-> QA review
```

Không coi tài liệu đã update là approved cho đến khi QA review report này.
