# Test Plans

Lưu test plan đã tạo từ requirement.

## File chính hiện tại

Test plan master cho toàn bộ dự án:

```text
qa-ai-workflow/test-plans/riffables-master.test-plan.md
```

Đầu vào chính:

```text
qa-ai-workflow/requirements/riffables-master.requirements.yaml
```

Sau khi review test plan này, bước tiếp theo là tạo:

```text
qa-ai-workflow/test-cases/riffables-master.test-cases.md
```

Gợi ý format:

```text
<feature-name>.test-plan.md
```
